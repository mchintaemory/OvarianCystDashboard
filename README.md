# OvarianCystDashboard
Thesis plots estimating prevalence ratios for PBB-153 and summed PCBs exposure in the Michigan PBB Registry
# Dashboard Features
* PBBs and PCBs forest plots with associated prevalence ratios for ovarian cysts
* Spline plots denoting nonlinear relationships between exposure levels and prevalence ratios
# Thesis Description
This project used deidentified data from the Michigan PBB Registry, with 396 women who self-reported whether they had clinically diagnosed ovarian cysts
or ovarian cysts requiring surgery. PBB-153 and PCB levels were taken from blood tests which were done between 2012-2022. 
# Takeaway
The results from this project are important to deepen physician and participant awareness when it comes to ovarian health and exposure levels.
